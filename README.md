## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/tpreisig/dark-bright-toggle.git
   ```
2. Navigate to the project directory:
   ```bash
   cd dark-bright-toggle
   ```
3. Run the code:
   ```bash
   open ./index.html
   ```
